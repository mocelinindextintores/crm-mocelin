# Publicar CRM Mocelin no GitHub e Vercel

## 1. GitHub
- Crie um repositório chamado `crm-mocelin`
- Envie os arquivos desta pasta

## 2. Vercel
- Entre no Vercel
- Clique em **Add New > Project**
- Escolha o repositório do GitHub
- Clique em **Deploy**

## 3. Observação
Para o deploy definitivo, substitua o `index.html` deste pacote pela versão final do CRM.

## 4. Próximo passo
Depois do deploy, conectar ao Supabase.
